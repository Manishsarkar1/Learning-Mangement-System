Learnly LMS

A role-based learning management system for students, instructors, and admins. It brings courses, assignments, quizzes, announcements, uploads, and analytics into one unified platform.

Overview

Learnly LMS is built to streamline the interaction between educators and learners. The system supports course browsing, enrollment, assignment submissions, quiz attempts, announcements, notifications, and profile management.

The backend is powered by Node.js, Express, and MySQL, while the frontend uses static HTML, CSS, and JavaScript pages served from frontend/public/.

Designed for a clean role-based workflow:

  • Students can enroll, submit work, and track progress.
  • Instructors can create content, grade submissions, and post updates.
  • Admins can manage users, permissions, analytics, and audit logs.

Getting Started
# Install dependencies
$ npm install

# Copy the environment file
$ Copy-Item .env.example .env

# Create the database in MySQL
mysql> CREATE DATABASE learnly_lms CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# Seed demo data and schema
$ npm run seed

# Start the app
$ npm start

# Optional development mode
$ npm run dev

User Roles
Role Capabilities
Admin Manage users, courses, permissions, analytics, and audit logs.
Instructor Create courses, add materials, post announcements, create quizzes, and grade submissions.
Student Browse courses, enroll, submit assignments, take quizzes, and view progress.
Key API Routes

POST /api/auth/login - sign in and receive a JWT cookie.

GET /api/courses - list available courses.

POST /api/submissions - submit assignment text or files.

POST /api/quizzes/:id/attempt - submit a quiz attempt.

GET /api/admin/analytics - view admin analytics.
